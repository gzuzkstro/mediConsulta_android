Esta aplicación está diseñada para trabajar en conjunto con https://github.com/Kahnos/mediConsulta

Por los momentos es necesario colocar manualmente la IP correspondiente al servidor antes de compilar para que la comunicación pueda ocurrir

Cuando la IP del servidor es correcta, se mostrará la lista de citas correspondeinte a la selección que se haga sobre el calendario. Los días ocupados están marcados
